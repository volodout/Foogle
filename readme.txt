Как пользоваться программой:
1) Для начала зайдите в командную строку и введите директорию, где расположена программа.

Пример: cd C:\Programs\Project\Python\Foogle\

2) Далее можно запустить программу. Для запуска сначала укажите эту строчку: python main.py
